LIVE

ENHANCEMENTS & FEATURES

Test Webcams, QR and Barcode Scanners

A range of camera use cases using Image Injection, from testing webcams to QR and Barcode scanning can now be tested on Windows Edge as well. Supported platforms include Desktop - Chrome on Windows & MacOS, Edge on Windows, and Mobile - Chrome on Android.

Try now

See more updates in Live Release Notes.

AUTOMATE

ENHANCEMENTS & FEATURES

Upload and Access Custom Media Files on iOS

You can now test scenarios that require uploading and accessing media files (images, video or audio) on Automate. Before executing your tests, upload these test files to BrowserStack using the REST API and then use them in your tests.

Learn more

Support for Cypress 10.8

Automate now supports all the new features that Cypress 10.8 introduces. Just ensure that your file and folder structure adheres to what Cypress 10.8 expects, and start testing on Automate.

Learn more

See more updates in Automate Release Notes.

APP AUTOMATE

ENHANCEMENTS & FEATURES

Support for Executing ADB Commands

App Automate now supports executing ADB commands such as logging firebase events, setting system-level preferences, getting diagnostic input via dumpsys commands, etc.

Try now

Support for Adding Contacts to Devices and Pre-Loaded Contacts

Now add contacts to our devices by uploading a .vcf file via the browserstack.uploadMedia caps in Appium and uploadMedia parameter in Espresso & XCUITest. You can also access pre-loaded contacts on devices.
